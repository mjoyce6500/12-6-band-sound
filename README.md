# 12-6 Band — Live Sound Engineer Quick Guide

A concise, practical guide to mixing and managing live audio for the 12-6 band. Includes references for the Behringer X32 Compact where console scenes and backups live.

## Quick Start (First 10 minutes)
- Confirm stage plot and input list.
- Power-up order: FOH → Monitors → Outboard → DIs.
- Static check: cables, mics, polarity, hum.
- Set gains to ~-18 dBFS and save an initial scene.

## Soundcheck Essentials
- Label channels and patch consistently.
- HPF 80–120 Hz on vocals; compress vocals 3:1 (fast attack).
- Build monitor mixes simple → personalize.
- Save two scenes: `Warmup` and `Show`.

## Behringer X32 Compact
Store console files in the behringer-x32-compact folder (configs, scenes, backups). Export full show scene and occasional backups after each gig. Reference: https://www.behringer.com (manufacturer) and your local console manual.

## Recording & Backups
- Record FOH stereo + stems if possible.
- Name files `YYYYMMDD_Venue_Song` and copy to two drives.

## Troubleshooting (Fast)
- No sound: check mute, gain, patch, cable.
- Feedback: reduce monitor level, notch offending freq.
- Thin vocals: mic placement, add presence around 2–4 kHz.
