# Behringer X32 Compact Mixer

The Behringer X32 Compact is a digital mixing console designed for live sound applications. It offers a wide range of features including 32 channels, 16 mix buses, and a user-friendly interface, making it suitable for both small and large venues. The compact design allows for easy transport and setup, while still providing professional-grade audio quality.

## Configurations Saved in This Folder

This folder contains various configurations and settings for the Behringer X32 Compact mixer, organized as follows:

- **configs/12-6_Warmup.x32**: Configuration for the warmup session, optimized for sound checks and initial setups.
- **configs/12-6_Show.x32**: Configuration for the live show, tailored for full band performances with specific channel settings and effects.
- **configs/backups/backup_YYYYMMDD.x32**: Backup files of the mixer configuration, saved with a date format for easy identification.
- **configs/scenes/Warmup.scn**: Scene settings for the warmup session, including channel settings and effects.
- **configs/scenes/Live.scn**: Scene settings for the live performance, customized for the full band setup.
- **patches/input_patch.csv**: Input patching configuration in CSV format, detailing how inputs are routed on the mixer.
