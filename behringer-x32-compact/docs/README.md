# Behringer X32 Compact Mixer Documentation

## Overview

The Behringer X32 Compact is a digital mixing console designed for live sound applications. It features a user-friendly interface, extensive routing options, and high-quality audio processing capabilities. The compact design makes it suitable for smaller venues while still providing professional-grade performance.

## Configurations Saved in This Folder

This folder contains various configurations and settings for the Behringer X32 Compact mixer, including:

- **Warmup Configuration:** `configs/12-6_Warmup.x32` - This file contains the saved configuration for the warmup session, ensuring a smooth start to the performance.
- **Live Show Configuration:** `configs/12-6_Show.x32` - This file contains the saved configuration for the live show session, optimized for the full band setup.
- **Backup Configurations:** `configs/backups/backup_YYYYMMDD.x32` - Regular backups of the mixer configuration are saved here, allowing for easy restoration if needed.
- **Scene Settings:** 
  - `configs/scenes/Warmup.scn` - Contains scene settings for the warmup session, including channel settings and effects.
  - `configs/scenes/Live.scn` - Contains scene settings tailored for the live performance.
- **Input Patching:** `patches/input_patch.csv` - Details how inputs are routed on the mixer in CSV format.
